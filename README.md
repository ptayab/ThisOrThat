# ThisOrThat
A Decision Making Web Application

## Getting started
```
cd repo_folder
git clone https://git.cs.usask.ca/370group_team20/equipay.git
```

## Running the Project
Backend:
```
cd existing_repo
cd ./backend
npm install
npm start
```

Frontend:
```
*open a second terminal
cd existing_repo
cd ./equi-pay-app
npm install
npm start
```
