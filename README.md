# ThisOrThat
A Decision Making Web Application

## Getting started
```
cd repo_folder
git clone https://github.com/ptayab/ThisOrThat.git
```

## Running the Project
Server:
```
cd existing_repo
cd ./server
npm install
npm start
```

Frontend:
```
*open a second terminal
cd existing_repo
cd ./this-or-that
npm install
npm start
```
